# MCP Servers: Web3 Industry

## Core MCP Servers

### 1. Blockchain MCP Server
**Data Sources:** Ethereum, Polygon, Arbitrum, Solana nodes
**Protocols:** JSON-RPC, WebSocket
**Agents Using:** All agents

### 2. Web3 MCP Server (Price/Market Data)
**Data Sources:** Coingecko, DEX aggregators, Chainlink oracles
**Update Frequency:** Real-time
**Agents Using:** DeFi Analyst, Security Monitor

### 3. IPFS MCP Server
**Data Sources:** IPFS nodes, Pinata, NFT.storage
**Operations:** Upload, pin, retrieve metadata
**Agents Using:** NFT Manager

### 4. Security Tools MCP Server
**Tools:** Slither, Mythril, Echidna (fuzzing)
**Agents Using:** Smart Contract Auditor, Security Monitor

### 5. DAO MCP Server
**Platforms:** Snapshot, Tally, Aragon
**Agents Using:** DAO Coordinator

### 6. DEX MCP Server
**Protocols:** Uniswap, Curve, Balancer APIs
**Agents Using:** DeFi Analyst

### 7. NFT Marketplace MCP Server
**Marketplaces:** OpenSea, Rarible, LooksRare
**Agents Using:** NFT Manager

### 8. Oracle MCP Server
**Providers:** Chainlink, Band Protocol
**Agents Using:** DeFi Analyst, Smart Contract Auditor

## Infrastructure
- **RPC Providers:** Alchemy, Infura, QuickNode
- **Indexing:** The Graph, Moralis
- **Cost:** $1.5K-$3K/month (Phase 1), $4K-$7K/month (Phase 3)
